Alguns seletores CSS

Em CSS, seletores são padrões usados para selecionar o(s) elemento(s) que você desejar estilizar.

Seletor universal

É representado por um asterisco "*", ele pode ser usado para selecionar todo e qualquer tipo de elemento em uma página.

Seletor por Tag/Elementos

Usado para selecionar elementos por seus nomes tags HTML.

Selecionado com classe!

A classe é o seletor mais comum e fácil de utilizar, é possível reaproveitá-lo em qualquer elemento da página e é altamente recomendado em qualquer projeto e é identificado por um "." mais "nome da class"

Selecionando com o id do elemento

Este seletor tem como alvo um elemento com um id específico e é identificado por um "#" mais "nome do id"

Seletores de atributos

É usado para selecionar elementos com um atributo especificado

Pseudo-classes

Uma pseudo-classe é usada para definir um estado especial de um elemento.

  • Estilize um elemento quando um usuário passar o mouses sobre ele;
  • Links visitados com um estilo e não visitados de forma diferente.

Pseudo-elementos

Um pseudo-elemento é usado para estilizar uma parte específica do elemento selecionado.

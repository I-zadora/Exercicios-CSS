CSS

O CSS é uma linguagem de estilização, ou seja, ela é utilizada para definir como os documentos escritos em uma linguagem de marcação devem ser apresentados em termos de formatação e layout.

Quando usado junto do HTML, o CSS é a linguagem que utilizamos para customizar as cores, tamanhos, bordas e outros atributos dos elementos da página.

Modelo de caixa

O modelo de caixa CSS é um contêiner que contém várias propriedades, incluindo bordas, margem, preenchimento e o próprio conteúdo.

Ele é usado para criar o design e layout de páginas da Web. Ele pode ser usado como um kit de ferramentas para personalizar o layout de diferentes elementos.
